# 🔬 오늘연구소

한국어 심리테스트·운세·건강·금융 계산기를 모아놓은 정적 웹 포털입니다.  
오늘 하루를 더 재미있게 보낼 수 있는 테스트와 계산기를 한 곳에서 제공합니다.  
별도 백엔드 없이 순수 HTML + CSS + Vanilla JS로 구성되어 있어 GitHub Pages로 즉시 배포 가능합니다.

---

## 주요 기능

### 운세
| 테스트 | 설명 |
|---|---|
| 오늘의 운세 | 생년월일로 알아보는 애정·금전·건강·직장운 |
| 별자리 운세 | 12별자리별 오늘의 운세 완전 분석 |
| 띠별 운세 2025 | 12띠별 한 해 운세 |
| 사주 간이 분석 | 생년월일시 기반 오행 밸런스 및 2025년 운세 |
| 2026 신년 운세 | 새해 재물·애정·건강·직업운 분석 |
| 오늘의 타로 | 과거·현재·미래 3장 카드 리딩 |
| 꿈 해몽 | 장소·대상·분위기 선택으로 꿈 풀이 |
| 혈액형 궁합 | 16가지 조합별 상세 궁합 분석 |
| 내 이름 점수 | 획수·수리로 보는 이름 운세 |
| 수비학 행운 숫자 | 이름과 생년월일로 계산하는 행운 숫자 |

## 기술 스택

- **HTML5** — 각 테스트별 독립 페이지
- **CSS3** — 다크 테마 기반 반응형 디자인, CSS 변수, 애니메이션
- **Vanilla JavaScript** — 프레임워크 없이 순수 JS로 구현
- **Google Fonts** — Noto Sans KR

> 외부 API·백엔드·데이터베이스 의존성 없음 — 정적 파일만으로 동작

## 배포

GitHub Pages 기준:

1. 이 저장소를 Fork 또는 Clone
2. GitHub 저장소 **Settings → Pages → Source** 에서 `main` 브랜치 루트(`/`) 선택
3. 자동 생성된 URL로 접속
